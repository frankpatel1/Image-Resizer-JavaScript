# Image Resizer JavaScript

## 🖼️ Overview

Image Resizer JavaScript is a simple and efficient web-based tool that allows users to resize images directly in their browser without requiring server-side processing. Built using HTML, CSS, and JavaScript, the application leverages browser-based image processing techniques to quickly modify image dimensions and generate resized outputs.

This project is ideal for learning image manipulation, file handling, and canvas-based processing in JavaScript. Browser-based image resizing commonly uses HTML Canvas APIs for client-side image processing and scaling.

## ✨ Features

* Upload images from your device
* Resize images by custom dimensions
* Real-time image preview
* Maintain aspect ratio support
* Fast client-side processing
* Download resized images instantly
* Responsive user interface
* No external dependencies required

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* HTML5 Canvas API

## 🚀 Getting Started

### Prerequisites

You only need a modern web browser:

* Google Chrome
* Mozilla Firefox
* Microsoft Edge
* Safari

### Installation

Clone the repository:

```bash
git clone https://github.com/frankpatel1/Image-Resizer-JavaScript.git
```

Navigate to the project folder:

```bash
cd Image-Resizer-JavaScript
```

Open `index.html` in your preferred browser.

## 📖 Usage

1. Launch the application.
2. Upload an image file.
3. Enter desired width and height values.
4. Preview the resized image.
5. Download the processed image.
